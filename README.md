# Smart Shuttle AI System

This project is a comprehensive AI-powered system for managing a smart shuttle service. It includes modules for passenger counting, driver behavior monitoring, data preprocessing, backend services, and frontend applications for students, drivers, and administrators.

## Project Structure

```
smart-shuttle-ai-system/
├── ai_models/
│   ├── passenger_counting/
│   └── driver_behavior/
├── preprocessing/
│   ├── passenger_vision/
│   └── driver_vision/
├── backend/
│   ├── api/
│   └── database/
├── frontend/
│   ├── student_app/
│   ├── driver_app/
│   └── admin_dashboard/
├── analytics/
│   └── revenue_engine.py
├── docs/
├── .gitignore
└── README.md
```
